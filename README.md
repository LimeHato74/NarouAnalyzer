# Narou Analyzer Pro V4

Web小説（なろう・カクヨム等）の執筆支援ツールです。
「自分の小説、客観的に見てどうなんだろう？」という疑問を解消するために作りました。

恋愛、ファンタジー、SFなど**18種類のジャンル**に合わせて、文章の「読みやすさ」や「構成バランス」を自動でスコアリングします。

![License](https://img.shields.io/badge/license-MIT-green)

## どんなツール？

ただの文字数カウントではありません。ジャンルごとの「お約束」や「傾向」を元に、以下のような視点で分析します。

1.  **ジャンル別採点**
    * 「異世界恋愛なら会話多め」「純文学なら情景描写重視」など、選択したジャンルによって評価基準が自動で切り替わります。
2.  **「壁」の検出**
    * スマホで読みにくい「改行のない長文の塊」を警告します。これだけでブラウザバック率が変わります。
3.  **感情の可視化**
    * 物語の盛り上がり（ポジティブ/ネガティブ）をグラフ化。ストーリーが単調になっていないかチェックできます。
4.  **具体的なアドバイス**
    * 「漢字が多すぎる」「一文が長くてリズムが悪い」など、修正すべきポイントをピンポイントで指摘します。

## 使い方

面倒なインストール作業は不要です。

1.  このリポジトリの `index.html` をダウンロードする。
2.  PCやスマホのブラウザ（Chrome, Safariなど）で開く。
3.  本文をコピペして「解析」ボタンを押すだけ。

> **安心設計**
> 解析はすべてブラウザ上のJavaScriptで行われます。執筆中の本文が外部サーバーに送信されることは一切ないので、非公開の原稿でも安心して使ってください。

## 対応ジャンル

以下のカテゴリごとに、目標とする「会話比率」や「文体リズム」が調整されています。

* **恋愛**（異世界 / 現実世界）
* **ファンタジー**（ハイファン / ローファン）
* **文芸**（純文学 / ホラー / 推理 / 歴史 等）
* **SF**（宇宙 / VRゲーム / パニック 等）
* **その他**（童話 / 詩 / リプレイ / エッセイ）

## 開発者向け情報 (Tech Stack)

カスタマイズしたい方向けの情報です。CDN依存なので、ローカルサーバーを立てなくても `file://` で動きます。

* **Vue.js 3** (Composition API)
* **Tailwind CSS + DaisyUI** (UIコンポーネント)
* **Chart.js** (レーダーチャート・感情曲線)
